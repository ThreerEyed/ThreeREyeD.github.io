    array

    概念：

    ​ 是多个相同数据类型，按照一定顺序进行排列的集合。用一个名字来命名，通过编号的形式对数据进行统一管理。

    特点：

    ​ 1、数组属于应用数据类型。数组内的元素可以是任意数据类型，可以是基本类型也可以是引用数据类型

    ​ 2、数组的长度一定，且不能随意更改即数组不可变

    ​ 3、数组有序，因为它是有下标的和python的列表类似，但是列表可变

    ​ 4、创建数组会在内存上开辟一段连续的内存空间

    ​ 5、通过下标访问数组内元素

    分类：

    ​ 按照维度分类：一维数组、二维数组

    ​ 按照数组内元素分类，基本数据类型数组、引用数据类型数组

    数组的声明和使用：

    数组的创建：

    数组需要先声明，再初始化

    数组的初始化分为静态初始化和动态初始化

    静态初始化是指初始化和赋值同时操作

    动态初始化是指先创建一个固定长度的数组，然后再进行赋值

    通过下标的形式来获取

    获取数组的长度：

    通过数组的属性length来获取

    数组的遍历：

    通过for循环来遍历数组，和python一样

    数组元素的初始默认值：
    基础数据类型
    1. 整型（byte、short、int、long）：0
    2. 浮点（float、double）：0.0
    3. char：0
    4. boolean：false
    引用数据类型
    1. 数组：null
    2. String：null
    3. 接口： null
    数组的内存解析（数组在内存中的具体体现）:

    声明数组的时候会在内存中的栈中存储一个变量，在堆中开辟一块连续的初始值为默认的内存，然后该段内存会有一个地址，地址的值会储存在栈中的变量中，这个地址直接指向堆中的对应的内存。

    当栈中的变量指向一个新的堆中的数组的时候，原来的数组就会被垃圾回收掉，当局部变量不再被使用的时候，栈中的变量会出栈，堆中的数组没有被变量指向的时候，也会被垃圾回收。

    2. 每行数字左右对称，由1开始逐渐变大。

    4. 每个数字等于上一行的左右两个数字之和。可用此性质写出整个杨辉三角。即第n+1行的第i个数等于第n行的第i-1个数和第i个数之和，这也是组合数的性质之一。即 C(n+1,i)=C(n,i)+C(n,i-1)。
